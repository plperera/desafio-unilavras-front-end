Um pouco Sobre Mim

Me chamo Pedro e sou formado em Engenharia de Software pela Driven Education, onde obtive experiência em +25 projetos e o mais importante criei minha Paixão pela área. Junto a isso, estou sempre expandindo meus conhecimentos, e agora estou aprendendo em Java e Spring Boot, e estou totalmente aberto a colaborações e oportunidades. Fique à vontade para entrar em contato comigo!
Atualmente estou cursando Analise e Desenvolvimento de Sistemas e trabalho como Freelancer, desenvolvendo desde projetos mais complexos a projetos simples. No momento atual uso as seguintes tecnologias: React.js, Node.js, TypeScript, Docker, PostgreSQL, PrismaORM, nginx, design UX/UI e me foco na satisfação do cliente através de designs responsivos e gerenciamento eficaz de projetos.

Meus Projetos e Experiências

Freelancer - NickTeAjuda

Desenvolvi autonomamente um sofisticado Ecommerce totalmente responsivo e equipado com um Painel Administrativo inovador, concebido para conferir maior autonomia ao cliente e otimizar a gestão do site. Através de um robusto sistema CRUD, possibilitou-se o gerenciamento completo de produtos, categorias, imagens (com armazenamento via Firebase), e métodos de entrega. Realizei ainda a customização do layout da página inicial, gerenciamento de pedidos e implementação de um gateway de pagamentos seguro via MercadoPago.

Freelancer - UpFitness

Desenvolvi um E-commerce, empregando React juntamente com Styled Components para criar uma interface de usuário elegante e responsiva no Front-end, enquanto utilizei Node.js e Express para estruturar um Back-end eficiente. lém disso, desenvolvi uma solução que permite aos clientes serem direcionados diretamente para o WhatsApp do vendedor, com uma mensagem personalizada que solicita um orçamento baseado nos itens selecionados no carrinho de compras. Esta funcionalidade melhora significativamente a experiência do usuário e facilita a comunicação rápida e eficaz entre o cliente e o vendedor.

Case Técnico - WePass

Desenvolvi um software de gerenciamento de senhas e credenciais, utilizando React e Styled Components para desenhar uma interface de usuário elegante e adaptável no front-end. No back-end, empreguei Node.js e Express para garantir eficiência. Uma característica destacada deste projeto é a criptografia robusta, aplicada a todos os dados sensíveis, como senhas e números de cartão. Além disso, organizei as credenciais em categorias: Dados do Cartão, Login e Notas. Esta última foi criada para acomodar informações que não se encaixavam nas categorias anteriores

Projeto em Grupo - Driven.t

Desenvolvemos um Software de gerenciamento de eventos!! Lidar com código legado foi um dos grandes desafios, e, para aumentar a segurança, pela primeira vez realizei testes de integração utilizando Jest. Usamos o Scrum como nossa metodologia, fazendo uso do Trello para estruturar melhor nossas sprints. Ao final da terceira sprint, não só atendemos aos requisitos básicos, mas também incorporamos funcionalidades extra que potencializaram nosso software.

Projeto em Grupo - Linkr

Em colaboração com minha equipe, lançamos o Linkr, uma inovadora rede social centrada no compartilhamento de links. Além das funcionalidades tradicionais como trending de hashtags e pesquisa de perfis, os usuários poderiam "repostar" links, similar a um "retweet", bem como curtir, comentar e visualizar quem engajou com suas postagens. A plataforma também extraía metadados dos links compartilhados, enriquecendo a discussão e contextualizando o conteúdo.

Projeto em Grupo - BuzzQuizz

Junto ao meu grupo, desenvolvemos um projeto front-end: um site interativo de Quiz. Baseado nas respostas dos usuários, o sistema calculava uma pontuação e apresentava resultados personalizados, como "Quão Grifinória de Harry Potter você é". Além de consumir dados de uma API, nosso site permitia que qualquer usuário criasse e compartilhasse seu próprio quiz, reforçando a interatividade e engajamento da plataforma.
